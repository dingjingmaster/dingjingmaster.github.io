go import 下划线

在 Golang 里，import 的作用主要是导入其它 package，如果 import 后加下划线，则表示，并不导入包，只是执行包里的 init() 函数 如:

import (
	"github.com/astaxie/beego"
	_ "iyuedu_service/routers"
)

go import 别名

import mongo "mywebapp/libs/mongodb/db"
// 为 "mywebapp/libs/mongodb/db" 包取别名为 "mongo"

go import 点操作

import . “xxx/lib”
// 点操作对包 lib 的调用直接省略包名。
