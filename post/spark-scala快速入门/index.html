Spark Scala快速入门

SPARK

1、spark参数配置 sparkConf类

  • 配置spark程序，以key-value的形式设置spark参数。它的设置优先于系统设置。
  • 该类所有方法都支持链接:new SparkConf().setMaster(“local”).setAppName(“My app”)
import org.apache.spark.SparkConf
val conf = new SparkConf().setAppName("test").setMaster("local")

2、初始化spark SparkContext类

import org.apache.spark.SparkContext
val sc = new SparkContext(conf)

3、读取hadoop文件

val data = sc.textFile("hadoopPath")

4、rdd 操作

1. Transformation

  • map(func) 源的每个元素通过func函数返回一个新的rdd(分布式数据集)
  • filter(func) 源的每个元素通过func,使func返回 true 的元素返回成新的rdd
  • flatMap(func) 类似map,但每个输入返回0到多个输出，func要求返回一个Seq
  • mapPartitions(func) 类似map,map是每个函数调用一下func,mapPartition是每个分块调用一次func,func必须是 Iterator<T> => Iterator<U> 输入是迭代器，返回也是迭代器
  • mapPartitionWithIndex(func)
  • sample(whithReplacement,fraction,seed) 使用给定随机数发生器种子，对数据的一小部分进行采样。三个参数。
  • union(otherrdd) 根据合并两个rdd返回一个新的rdd，新rdd包含两个旧rdd的元素,且是去过重的
  • intersection(rdd) 返回两个rdd的交集
  • distinct([numTasks]) 返回一个新的rdd,新rdd的元素是旧rdd中去过重的.
  • groupByKey([numTasks]) 数据集为(k,v)时,返回(k,Iterable)将k相同的合并成一个链表.
  • reduceByKey(func,[numTasks]) 数据集为(k,v)时,返回(k,v),将k相同的元素都通过func函数处理,func函数必须是func(v,v)。
  • aggregateByKey(ZeroValue)(seqOp, combOp, [numTasks]) 对(k,v)中k相同的值进行聚合操作，在聚合中使用一个初始值ZeroValue,seqOp在同一个partition中进行聚合,combOp是在不同partition中进行聚合.
  • sortByKey([ascending],[numTasks]) 作用于(k,v)形式的rdd，并对k进行排序。ascending 排序后RDD中元素是升序还是降序(默认true升序)
  • join(otherRDD,[numTasks]) 当(k,v)和(k,w)数据集被调用时，返回每个键的所有元素对(k,(v,w))
  • cogroup(otherRDD, [numTasks]) 将多个RDD中同一个key对应的value组合在一起，最多可以组合4个RDD返回 (k,(Iterable<V>,Iterable<W>))
  • cartesian(otherRDD) 当前RDD和otherRDD每个元素进行笛卡尔计算的结果
  • pipe(command,[envVars]) 通过shell管理rdd每个分区,per或bash脚本.RDD元素被写入进程的stdin,输出到stdout的行被作为字符串的RDD返回。
  • coalesce(numPartitions) 减少RDD中的分区数量为numParitions,用于在过滤大型数据集后更高效地运行操作
  • repartition(numPartitions) 随机调整RDD中的数据以创建更多或更少的分区,这会通过网络对所有数据洗牌。
  • repartitionAndSortWithinPartition(partitioner) 在repartition重新分区之后,还要进行排序,建议直接使用reparationAndSortWithinParitions算子。该算子可以一边重新分区一边排序,相比先重新分区再排序性能要高.

2. Action

  • reduce(func) 使用函数func(接受两个参数,返回一个)聚合数据集元素.
  • collect() 将数据集的所有元素作为数组返回
  • count() 返回数据集中元素的数量.
  • first() 返回数据集中第一个元素,相当于take(1)
  • take(n) 以数组的形式,返回数据集中前n个元素
  • takeSample(withReplacement, num, [seed]) 随机返回num个元素的数组
  • takeOrdered(n, [ordering]) 按照排序(默认升序)返回RDD前n个元素
  • saveAsTextFile(path) 将RDD保存为text文件
  • saveAsSequenceFile(path) 将RDD以SequenceFile的文件格式保存在HDFS上
  • saveAsObjectFile(path) 将RDD中的元素序列化成对象,存储到文件中
  • countByKey() 对于(k,v)形式的RDD,通过该函数以(k, Int)返回每个k的个数
  • foreach(func) func 传入参数是每个partition,每次foreach得到一条数据,处理一条数据.
