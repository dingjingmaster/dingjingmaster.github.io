Ubuntu卸载vmware

2018-10-11
环境配置

        • 1. 查看安装的虚拟机
        • 2. 卸载虚拟机

1. 查看安装的虚拟机

vmware-installer -l

2. 卸载虚拟机

sudo vmware-installer --uninstall-product vmware-workstation

然后根据提示看看是否保存虚拟机配置文件输入 yes 或 no 即可。
