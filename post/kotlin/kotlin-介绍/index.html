Kotlin 介绍

2018-11-09
kotlin

      • Kotlin 的优缺点
      • Kotlin 安装
      • Kotlin 架构
        • Kotlin 以Java为目标

Kotlin是一种新的开源编程语言，如Java、JavaScript等。它是一种高度静态类型的语言，将功能和技术结合在一起。它在JVM上运行。Kotlin受到Java、Scala、Groovy、Gosu等其他编程语言的影响。Kotlin为开发提供了互操作性、代码安全性和透明性。

Kotlin 的优缺点

优点： 1. 简单的语言： Kotlin是一种函数式语言,很容易学习。语法与Java非常相似，因此很容易记住。Kotlin具有更强的表达能力，使代码更易于阅读和理解。 2. 简洁： Kotlin 基于 JVM 和 函数式编程，因此代码更加简洁。 3. 运行时和性能： Kotlin 有更好的性能以及占用资源更少的运行时。 4. 交互性 5. 更新： Kotlin = Java + 更新的特性

缺点： 1. 命名空间声明： Kotlin允许开发人员在顶层声明函数。然而，每当在应用程序的许多地方声明相同的函数时，就很难理解调用的是哪个函数。 2. 没有静态声明： Kotlin没有Java这样的静态处理修饰符，这可能会给传统的Java开发人员带来一些问题。

Kotlin 架构

Kotlin 以Java为目标

Kotlin是一种编程语言，它有自己的体系结构来分配内存并为最终用户生成高质量的输出。下面是Kotlin编译器将以不同的方式工作的不同场景，当它针对不同的其他语言(如Java和JavaScript)时。Kotlin编译器创建一个字节代码，该字节代码可以在JVM上运行，这正好等于Java .class文件生成的字节代码。当两个字节编码的文件在JVM上运行时，它们可以相互通信，这就是Kotlin中如何建立互操作特性的方法。

每当Kotlin以JavaScript为目标时，Kotlin编译器都会将.kt文件转换为ES5.1，并为JavaScript生成兼容的代码。Kotlin编译器能够通过LLVM创建平台基础兼容的代码。
