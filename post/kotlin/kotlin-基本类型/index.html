Kotlin 基本类型

Kotlin编程语言中的基本数据类型。

数字类型

Kotlin中的数字表示与Java非常相似，但是Kotlin不允许内部转换不同的数据类型。下表列出了不同数字的不同可变长度。

类型 大小
Double 64
Float 32
Long 64
Int 32
Short 16
Byte 8 
fun main(args: Array<String>) {
   val a: Int = 10000
   val d: Double = 100.00
   val f: Float = 100.00f
   val l: Long = 1000000004
   val s: Short = 10
   val b: Byte = 1

   println("Your Int Value is "+a);
   println("Your Double  Value is "+d);
   println("Your Float Value is "+f);
   println("Your Long Value is "+l);
   println("Your Short Value is "+s);
   println("Your Byte Value is "+b);
}

//
Your Int Value is 10000
Your Double  Value is 100.0
Your Float Value is 100.0
Your Long Value is 1000000004
Your Short Value is 10
Your Byte Value is 1

字符类型

Kotlin使用char表示字符。字符应该声明在一个单引号，如 ‘c’。不能像数字变量一样声明字符变量。Kotlin变量可以用两种方式声明——一种是使用“var”，另一种是使用“val”。

fun main(args: Array<String>) {
   val letter: Char    // defining a variable
   letter = 'A'        // Assigning a value to it
   println("$letter")
}

//
A

布尔类型

布尔值和其他编程语言一样非常简单。布尔值只有两个值——真值和假值。

fun main(args: Array<String>) {
   val letter: Boolean   // defining a variable
   letter = true         // Assinging a value to it
   println("Your character value is "+"$letter")
}

//
Your character value is true

字符串类型

字符串是字符数组。与Java一样，它们本质上是不可变的。Kotlin中有两种字符串，一种是原始字符串，另一种是转义字符串。

fun main(args: Array<String>) {
   var rawString :String  = "I am Raw String!"
   val escapedString : String  = "I am escaped String!\n" // '\n' 是转义字符

   println("Hello!"+escapedString)
   println("Hey!!"+rawString)   
}
//
Hello!I am escaped String!

Hey!!I am Raw String!

数组

数组是同构数据的集合。与Java一样，Kotlin支持不同数据类型的数组。

fun main(args: Array<String>) {
   val numbers: IntArray = intArrayOf(1, 2, 3, 4, 5)
   println("Hey!! I am array Example"+numbers[2])
}
//
Hey!! I am array Example3

集合

集合是数据结构中一个非常重要的部分，它使软件开发对工程师来说变得很容易。Kotlin有两种类型的集合——一种是不可变的集合(这意味着不能编辑的列表、映射和集合)，另一种是可变的集合(这种类型的集合是可编辑的)。

fun main(args: Array<String>) {
   val numbers: MutableList<Int> = mutableListOf(1, 2, 3) // 可变列表
   val readOnlyView: List<Int> = numbers                  // 不可变列表
   println("my mutable list--"+numbers)        // prints "[1, 2, 3]"
   numbers.add(4) // 可以添加元素
   println("my mutable list after addition --"+numbers)        // prints "[1, 2, 3, 4]"
   println(readOnlyView)     
   readOnlyView.clear()    // 不可变列表不可以清空操作，会报错。
// gives error  
}

Ranges

range 用于遍历指定范围的数据

val i:Int  = 2
for (j in 1..4)
print(j) // prints "1234"

if (i in 1..10) { // 相当于 1 < = i && i < = 10
   println("we found your number --"+i)
}
}
