Hadoop 安装

2018-11-08
hadoop

      • 准备
      • 开始安装

准备

  • 系统：linux/ubuntu
  • java jdk 1.5 +
  • ssh
  • rsync
  • hadoop 稳定发行版

开始安装

  • 安装 java 配置 JAVA_HOME

    手动下载 java 安装包, tar 命令解压，配置好 JAVA_HOME

  • 安装 ssh 和 rsync

    sudo apt-get install ssh rsync

  • 解压 hadoop

  • ssh 无密码访问

    ssh localhost                       # 按下 ctrl + C 打断
cd ~/.ssh/
ssh-keygen -t dsa                   # 一直回车
cat ./id_dsa.pub >> authorized_keys # 加入授权, 我追加的形式把 公钥写入 authorized_keys 文件中验证不了,我直接覆盖的写 ">",用vim打开复制应该也可以，可能与 文件结尾EOF有关系。
chmod 0666 ./authorized_keys        # 修改权限

# 修改 sshd 配置
sudo vi /etc/sshd_config
# 修改以下几项(为了省事，可以直接把文件配置全部注释后复制以下配置)
  Port 22
  AddressFamily any
  ListenAddress 0.0.0.0
  StrictModes no
  PermitRootLogin yes
  RSAAuthentication yes
  PubkeyAuthentication yes
  AuthorizedKeysFile  .ssh/authorized_keys
  PasswordAuthentication no
  ChallengeResponseAuthentication no
  UsePAM yes
  X11Forwarding yes
  PrintMotd no
  AcceptEnv LANG LC_*
  Subsystem   sftp    /usr/lib/openssh/sftp-server
# :wq 保存即可
ssh localhost # 第一次要输入用户密码，之后就可以免密码连接了

  • 配置 hadoop 伪分布模式

    1、 配置 hadoop-env.sh 文件

    vi hadoop-env.sh
# 25 行替换为 export JAVA_HOME="java 安装路径"

    2、 配置 DFS

    # 1. 配置 core-site.xml
vi core-site.xml
# 配置为如下
<configuration>
    <property>
        <name>fs.defaultFS</name>
        <value>hdfs://localhost:9000</value>
    </property>
</configuration>

# 2. 配置 hdfs-site.xml
vi hdfs-site.xml
# 配置如下
<configuration>
    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
</configuration>

# 3. 格式化 DFS
hdfs namenode -format # 需要把 ${HADOOP_HOME}/bin/ 添加到 PATH 系统变量中

# 4. 测试 启动 NameNode 和 DataNode
sbin/start-dfs.sh # 需要到 ${HADOOP_HOME} 目录下
# 浏览器打开 http://localhost:50070/ 可以看到 NameNode 相关信息

    3、配置 YARN

    # 1. 配置 mapred-site.xml
<configuration>
    <property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
    </property>
</configuration>

# 2. 配置 yarn-site.xml
<configuration>
    <property>
        <name>yarn.nodemanager.aux-services</name>
        <value>mapreduce_shuffle</value>
    </property>
</configuration>

# 3. 启动 ResourceManager 和 NodeManager
sbin/start-yarn.sh
# 在浏览器中访问http://localhost:8088/，可以看到ResourceManager相关信息。

  • 此时查看 java 进程

    jps

  • hadoop 停止

    # 在 ${HADOOP_HOME}中执行
sbin/stop-dfs.sh
sbin/stop-yarn.sh
