PHP实现分页展示

2018-08-04
PHP

        • 一、思路
        • 二、实现

一、思路

  1. 要展示的几个字段
  2. 当前第 xx 页/共有 xx 页
  3. 首页
  4. 前一页
  5. 可以直接跳转到的页码
  6. 下一页
  7. 末页
  8. 共有多少条信息需要展示
  9. 针对当前页的页码需要进行特殊展示
  10. 提前需要知道的字段
  11. 当前是第几页
  12. 要展示信息的总数
  13. 一次展示的页数（前一页和下一页中展示出来的页码连接）
  14. 每页展示信息的条数
  15. 计算各个字段
  16. 共有 xx 页 （要展示数据总数 / 每页展示信息的条数）。注意:有小数点要加一。
  17. 首页 和 末页 分别写一个固定的 url，末页会用到总页数这一计算结果。
  18. 前一页 和 下一页，分别是当前页减1和加1，但是要注意边界值（页码取值要大于0小于总页码数）。
  19. 写好前端实现之后，后端查MySQL数据库时只要输出查询游标的偏移数和最大查询数即可得到请求页码的数据。

二、实现

以下是我php代码实现如果想直接用，改一下 <a> 标签的 href 属性即可

/* 分页功能 */
/**
 *  @$curPage:          当前页数
 *  @$totalNum:         总条数
 *  @$showPage:         一次显示页数
 *  @$everyPageNum:     每页展示条数
 */
function page_slip($curPage, $totalNum, $showPage, $everyPageNum) {
    $html = '';
    $totalPage = ceil($totalNum / $everyPageNum);
    if($curPage <= 1 || $curPage == '') {
        $curPage = 1;
    } else if ($curPage >= $totalPage) {
        $curPage = $totalPage;
    }
    $html .= '<li><a>第' . $curPage . '/' . $totalPage . '页  ' . '</a></li>';
    $html .= '<li><a href="/index.php/pageturn/0">' . '首页' . '</a></li>';
    $prePage = ($curPage <= 1) ? 1 : $curPage - 1;
    $nextPage = ($curPage >= $totalPage) ? $totalPage : $curPage + 1;
    $html .= '<li class="arrow"><a href="/index.php/pageturn/'.$prePage.'">&laquo;</a></li>';
    /* 展示显示分页 */
    $pageShowStart = (ceil($curPage / $showPage) - 1) * $showPage;
    $pageShowEnd = ($pageShowStart + $showPage) > $totalPage ?\
        $totalPage * 2 - $pageShowStart - $showPage - 1: $showPage;
    if ($showPage > $totalPage) {
        $showPage = $totalPage;
    } else if ($showPage > $pageShowEnd) {
        $showPage = $pageShowEnd;
    }
    /* 开始准备输出页数 */
    for($i = 1; $i <= $showPage; $i++) {
        $pageNow = $pageShowStart + $i;
        if($curPage == $pageNow) {
            $html .= '<li class="current"><a href="/index.php/pageturn/'\
              . $pageNow . '">' . $pageNow . '</a></li>';
        } else {
            $html .= '<li><a href="/index.php/pageturn/' . $pageNow . '">'\
              . $pageNow . '</a></li>';
        }
    }
    $html .= '<li><a href="/index.php/pageturn/' . $nextPage . '"> &raquo; </a></li>';
    $html .= '<li><a href="/index.php/pageturn/' . $totalPage . '">' . '末页' . '</a></li>';
    $html .= '<li><a>共' . $totalNum . '篇文章</a></li>';
    return $html;
}
